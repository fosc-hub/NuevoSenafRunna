# Registro de Cambios

En esta carpeta se deben guardar los documentos (archivos .txt, .md o capturas) que detallen las modificaciones realizadas en el código.

## Formato sugerido
- `FECHA_CAMBIO_DESCRIPCION.md`
- Ejemplo: `2026-02-25_correccion_estilos_navbar.md`

Al momento de entregar a otros técnicos, puedes incluir este registro para que sepan exactamente qué archivos fueron tocados.
